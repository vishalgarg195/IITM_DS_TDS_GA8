This analysis loads employee data, counts the Finance department employees, prints the result, and displays a histogram of department distribution.

Python Code Used


import pandas as pd
import matplotlib.pyplot as plt

# Load synthetic dataset (100 rows generated in ChatGPT conversation)
df = pd.read_csv("employee_data.csv")

# Count Finance employees
finance_count = df[df["department"] == "Finance"].shape[0]
print("Finance Department Count:", finance_count)

# Histogram
plt.figure(figsize=(8,6))
plt.hist(df["department"], edgecolor="black")
plt.title("Department Distribution")
plt.xlabel("Department")
plt.ylabel("Frequency")
plt.savefig("hist.png")
